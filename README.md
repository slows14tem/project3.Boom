# 폭탄 게임 제작
+ https://slows14tem.github.io/project3.Boom/

## 개요
자바 스크립트를 활용한 간단한 지뢰찾기 게임을 제작했다.
배열 생성과 셔플 기능을 사용해서 폭탄이 있는 칸을 피해서 모든 칸을 선택하면 성공!!
